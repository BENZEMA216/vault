# 故事短片-配乐方案

## ⚠️ 硬性约束（CRITICAL）

1. **仅输出推荐方案**：本技能只生成配乐推荐文档，不控制视频生成过程中的音频
2. **视频生成时无背景音乐**：视频生成时背景音乐已被主流程抑制，本技能不改变此行为
3. **用户偏好优先**：如用户有指定音乐风格，必须优先遵循
4. **文化适配**：配乐风格必须与视频类型一致（古风国风类用国风音乐，科幻未来类用电子音乐）

---

## 定位说明

本技能的作用是：在所有视频片段生成完成后，为用户输出一份**配乐推荐方案文档**。

该方案供用户后期手动添加背景音乐时参考，包含：
- 分段配乐风格与情绪建议
- 音效设计方案
- 音乐搜索关键词

**注意**：本技能不控制视频生成过程中的音频。视频生成时背景音乐已被主流程抑制。

---

## 输出格式

当为用户推荐配乐方案时，使用以下格式：

```
## 音乐方案推荐

### 整体风格
[描述整体音乐风格，如：温暖治愈的钢琴民谣风格]

### 分段配乐

**开场（0-10秒）**
- 风格：[如：极简钢琴，单音引入]
- 情绪：[如：孤独、安静]
- 参考：[如：久石让《Summer》前奏]

**发展（10-30秒）**
- 风格：[如：吉他加入，节奏渐起]
- 情绪：[如：希望、温暖]
- 参考：[如：民谣风格，类似《500 Miles》]

**高潮（30-45秒）**
- 风格：[如：弦乐进入，情感爆发]
- 情绪：[如：感动、幸福]
- 参考：[如：电影《忠犬八公》配乐风格]

**结尾（45-60秒）**
- 风格：[如：回归钢琴，渐弱收尾]
- 情绪：[如：余韵、温暖]
- 参考：[如：留白式结尾]

### 音效设计
- 环境音：[如：雨声、鸟鸣]
- 动作音：[如：脚步声、门开关]
- 情感音：[如：心跳声、呼吸声]

### 推荐搜索关键词
[提供 5-8 个搜索关键词，如：warm acoustic piano, emotional cinematic, gentle guitar]
```

---

## 工作流程

1. **输入分析**：读取 Storyboard 中的全局信息（故事线、情绪基调、类型）
2. **类型匹配**：确定故事类型 → 查阅下方对应的配乐指南
3. **分段设计**：按镜头组设计分段配乐方案（风格+情绪+参考）
4. **音效设计**：为关键画面设计音效层（环境音+动作音+情感音）
5. **关键词生成**：生成 5-8 个音乐搜索关键词
6. **方案输出**：按上方输出格式整合为完整方案文档

**输出物**：完整的配乐推荐方案文档
**完成标志**：方案已展示给用户

## 类型化情绪曲线配乐表

### 1. 温情治愈类

| 阶段 | 音乐特征 | 推荐乐器 |
|------|----------|----------|
| 孤独开场 | 缓慢、稀疏音符、小调 | 钢琴单音 |
| 相遇时刻 | 旋律渐起、温暖和弦 | 吉他+钢琴 |
| 情感升温 | 节奏加快、弦乐进入 | 弦乐+钢琴 |
| 温暖结局 | 大调、完整和弦、渐强 | 全乐队 |

### 2. 奇幻冒险类

| 阶段 | 音乐特征 | 推荐乐器 |
|------|----------|----------|
| 平凡世界 | 简单、日常感 | 木管乐器 |
| 冒险召唤 | 旋律跃起、节奏加快 | 铜管+打击乐 |
| 危机时刻 | 紧张、快速、不和谐音 | 全管弦乐 |
| 胜利归来 | 宏大、triumphant | 铜管+定音鼓 |

### 3. 科幻未来类

| 阶段 | 音乐特征 | 推荐音色 |
|------|----------|----------|
| 未来城市 | 节奏感、电子节拍 | 合成器Bass+鼓机 |
| 人机互动 | 温暖与冰冷对比 | 钢琴+电子音色 |
| 危机/冲突 | 低频震动、不规则节奏 | Sub Bass+工业噪音 |
| 情感升华 | 旋律化、加入人声 | 合成器+女声哼唱 |

### 4. 古风国风类

| 阶段 | 音乐特征 | 推荐乐器 |
|------|----------|----------|
| 山水开场 | 空灵、悠远、留白 | 古琴+箫 |
| 江湖恩怨 | 节奏鲜明、武侠感 | 二胡+打击乐 |
| 情感纠葛 | 婉转、哀怨、缠绵 | 琵琶+古筝 |
| 悟道/超脱 | 简化、回归宁静 | 单音箫+自然音 |

### 5. 恐怖悬疑类

| 阶段 | 音乐特征 | 推荐技法 |
|------|----------|----------|
| 日常假象 | 看似正常但略有违和 | 小调、轻微走音 |
| 不安渐起 | 节奏不稳、音高飘忽 | 弦乐滑音 |
| 恐怖高潮 | 突然静默或爆发 | 打击乐+尖叫音效 |
| 余悸未消 | 回归"正常"但留有伏笔 | 单音持续 |

## 音效设计通用指南

### 音效层级

```
第一层：环境底噪（持续存在，音量最低）
第二层：动作音效（跟随画面动作，音量中等）
第三层：情感音效（强调情绪，音量根据情绪强度变化）
第四层：音乐（贯穿全片，音量根据对话/旁白调整）
```

### 类型化音效参考

| 类型 | 环境音 | 动作音 | 情感音 |
|------|--------|--------|--------|
| 温情治愈类 | 鸟鸣/风声/雨声 | 脚步/门开关/餐具 | 心跳/呼吸 |
| 奇幻冒险类 | 神秘森林/洞穴回声 | 剑击/翅膀扇动 | 魔法能量/咒语 |
| 科幻未来类 | 太空舱/城市噪音/雨声 | 机械臂/飞行汽车 | UI界面/全息投影 |
| 古风国风类 | 流水/竹林沙沙/寺庙钟声 | 剑击/轻功衣袂/马蹄 | 古筝泛音/远处笛声 |
| 恐怖悬疑类 | 风声/地板吱嘎/远处异响 | - | 心跳加速/耳鸣/呼吸急促 |

## 配乐实用技巧

### 1. 节拍同步
- 音乐重拍与画面切换点对齐
- 动作音效与音乐节奏匹配

### 2. 情绪同步
- 音乐情绪领先画面情绪 1-2 秒（预告感）
- 高潮时刻音乐先于画面达到顶点

### 3. 对比运用
- 悲伤画面+平静音乐（克制更动人）
- 紧张画面+安静音乐（反差制造不安）

## 注意事项

1. **版权问题**：推荐使用免版权音乐（如 YouTube Audio Library、Free Music Archive）
2. **音量平衡**：音乐不能盖过对话/旁白，建议背景音乐音量在 -20dB 到 -15dB
3. **文化适配**：古风视频用国风音乐，科幻视频用电子音乐，保持风格统一
4. **用户偏好**：如用户有指定音乐风格，优先遵循用户要求

## 常见问题处理

**问题：用户对推荐的配乐方案不满意**
→ 询问用户偏好的音乐风格/情绪方向
→ 根据用户反馈调整方案，重新匹配类型

**问题：视频时长很短（≤15秒），不需要复杂的分段配乐**
→ 简化为单段配乐推荐：一个核心情绪 + 一种乐器/风格
→ 重点放在音效设计上

**问题：用户要求的音乐风格与视频类型不匹配**
→ 先确认用户意图（可能是有意的反差设计）
→ 如确实不匹配，委婉建议更合适的风格并说明原因
