---
name: task
description: 委托任务给子代理
---

将任务委托给专门的子代理处理。

**参数**：
- `agent` (必需): 子代理名称
- `prompt` (必需): 任务描述

**可用子代理**：
- `research`: 只读研究，搜索素材和参考
- `outline`: 只读规划，生成大纲和结构
- `general`: 通用执行，完整能力

**使用场景**：
- 并行处理多个独立子任务
- 需要深入研究某个具体问题
- 搜索范围不确定，需要广泛探索

**注意**：
- 子代理有独立上下文，任务描述要**完整自包含**
- 可以同时发起多个 task 调用（并行）
- 子代理结果需要你整合后呈现给用户
- 明确告诉子代理是「研究」还是「执行」
