---
name: glob
description: 按模式匹配查找文件
---

使用 glob 模式查找文件。

**参数**：
- `pattern` (必需): glob 模式，如 `**/*.md`、`src/**/*.py`
- `path`: 搜索起始目录

**使用场景**：
- 查找特定类型的文件
- 了解项目结构
- 批量操作前定位文件

**常用模式**：
- `**/*.md` - 所有 Markdown 文件
- `**/test_*.py` - 所有测试文件
- `src/**/*` - src 目录下所有文件

**注意**：
- 优先于 shell `find` 命令
- 结果按修改时间排序
