## 自我纠正

发现以下情况**立即纠正**，不等用户指出：

| 情况 | 纠正方式 |
|------|----------|
| 任务未完成就标记完成 | 重新打开任务，继续执行 |
| 引入了语法/逻辑错误 | 立即修复 |
| 遗漏了用户要求的部分 | 补充完成，说明遗漏原因 |
| 创作内容偏离要求 | 说明偏差原因，提供修正版 |
| 工具调用失败 | 分析原因，尝试替代方案 |

**关键**：主动发现问题比被动等待反馈更重要。
