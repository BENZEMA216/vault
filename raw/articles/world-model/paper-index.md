## 观察与发现

### 引用量趋势
- LeCun JEPA Vision (2022) 和 MuZero (2020) 占据最高引用（3000+），说明 world model 最受关注的两极：**认知架构理论** 和 **实际超人表现**
- Tree of Thoughts (3,634) 作为 LLM 规划方向引用量最高，说明 **LLM + world model 思维** 是当前学术热点
- Dreamer 系列 4 篇论文累计 4000+ 引用，是 world model RL 最成功的研究线

### 架构演进
- 2018-2020: VAE + RNN (Ha, DreamerV1)
- 2021-2023: Transformer (IRIS, STORM) + Discrete Representations (DreamerV2/V3)
- 2024-2025: Diffusion (DIAMOND, GameNGen, Sora) + JEPA (I-JEPA, V-JEPA)
- 2025-2026: Hybrid (LeWM, Genie 3, Cosmos) + SSM/Mamba 探索

### 产学融合
- Sora/Genie/Cosmos/GAIA 等产品级 world model 几乎都来自大公司（OpenAI/DeepMind/NVIDIA/Wayve）
- 学术界在理论（JEPA, Othello-GPT）和样本效率（EfficientZero, DIAMOND）上贡献更大
