## 按通信模式分类

### Debate/Discussion（辩论式）
- #1 Du et al. (1,429) — 奠基之作，多 Agent 辩论提升 factuality
- #6 Liang et al. (973) — 发散思维 + 评判者收敛
- #12 ReConcile (273) — 圆桌会议 + 异构 LLM 共识
- #10 MoA (321) — Mixture-of-Agents 层级聚合

### Role-Playing（角色扮演式）
- #5 CAMEL (1,194) — inception prompting 开创者
- #7 ChatDev (643) — chat chain + communicative dehallucination
- #4 MetaGPT (1,547) — SOP + publish-subscribe，最成熟
- #8 AgentVerse (549) — 动态角色调整

## 关键观察

1. **2023 年是奠基年** — 8 篇 S-tier 论文中有 7 篇来自 2023，它们定义了基本通信范式
2. **2024 年进入系统化** — 综述、平台、Scaling 研究密集出现
3. **2025 年的前沿突破** — Latent Space Communication 绕过自然语言，效率提升 2-471x
4. **Market-Based 是 Agent Economy 最直接相关的学术方向** — 但论文极少，机会巨大
5. **引用数与实际影响不完全对应** — AutoGen 引用虽高但社区评价 "breaking changes 太多"；CrewAI 无学术论文但商业化最成功
