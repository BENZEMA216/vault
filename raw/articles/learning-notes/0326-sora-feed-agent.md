# 对 AGENT 的影响是啥？
## AGENT 的价值
1. 4o 会覆盖 AGENT大部分/几乎全部的能力，我们要做一个**时间判断 （验证性价值）**
  1. openai 可能会吃到一定的品牌&技术红利（来源于数据&模型架构），但是他不在乎雕花/用户体验的事情
  1. AGENT 可能还是可以成为多模态的*插件*
1. 生成/理解模型不会为创作任务专门做优化，搞搞数据试试总还是有后训练的价值（先定义数据吧，怎么训再说）**（场景性价值）**
1. 即梦现在框架下，AGENT 会组合使用工具仍然是有意义的。**（总有工具不会被模型内化的）**




- 对结果的质量监督在 6 个月内 将没有任何意义了
- 对 AGENT 的知识补充好像也没啥意义了
- **创意**模块会特别重要，因为模型可以响应各种 Prompt，比如上面那种结构化的 Prompt 或者问题；核心是怎么找到一个场景下的创意表达
  - 为啥是创意？ 其实有点类似多模态领域的 Deep Research，就给用户提供更多的智力总量
- 多模态工具的使用，会在全模态统一之前始终发挥价值
- AGENT 对端内用户的理解和商业化，也很重要啊
