# 故事短片-创意与剧本

## 触发条件
当主流程进入 Phase 2（分镜设计）时加载本技能。

## 适用范围
- 用户有模糊想法/图片/视频，需要发展为完整故事剧本和分镜设计
- 专注于 15 秒 ~ 3 分钟叙事短片的剧本开发和分镜设计

## 不适用
- 纯产品广告（不涉及故事叙事）
- 教育类视频（按知识结构组织而非情感弧线）

---

## ⚠️ 硬性约束（CRITICAL）

1. 每个镜头组时长 **≤ 15 秒**（Seedance 2.0 限制）
2. 所有角色和场景设计必须**避免真实人脸**
3. **视觉叙事优先于对白**——能用画面表达的不用语言
4. 同一角色在不同场景中必须有明确的**视觉锚点**（特征锁定）
5. 剧本必须标注每场戏预计需要的镜头组数量

---

## 时长与剧本复杂度对照

| 时长 | 建议场次 | 角色数量 | 情绪转折 | 适合类型 |
|------|----------|----------|----------|----------|
| 15-30秒 | 1-2场 | 1-2个 | 1次 | 温情治愈类、恐怖悬疑类、反转 |
| 30-60秒 | 2-3场 | 2-3个 | 1-2次 | 温情治愈类、奇幻冒险类、古风国风类 |
| 1-3分钟 | 3-5场 | 3-4个 | 2-3次 | 奇幻冒险类、科幻未来类、复杂叙事 |

> **在开始创作前，先根据用户确认的时长查阅此表，确定剧本复杂度上限。**

---

## 核心创作哲学

### 1. 短片的叙事经济学
短片不是长片的压缩，而是独立的艺术形式：
- **单一核心驱动力**：聚焦一个情感内核或一个核心冲突，拒绝旁支蔓节
- **视觉即叙事**：能用画面交代的信息，绝不依赖对白或旁白
- **留白艺术**：给观众想象空间，不说透比说透更有力量

### 2. 情感弧线设计
每个故事都必须有清晰的情感曲线：
- **起点情绪**：观众最初感受到的基调（孤独/好奇/不安/温暖）
- **情绪转折点**：1-2个关键的情感转折时刻
- **情感余韵**：结尾后留在观众心中的感受（比情节更重要）

### 3. 可拍摄性思维
时刻记住剧本将被转化为视频：
- 每个场景都必须是"可见的"，避免抽象心理描写
- 考虑 Seedance 15 秒片段的承载能力——复杂动作需要拆解为多个镜头组
- 预判哪些元素会成为视觉亮点，哪些会成为一致性难题

## 故事类型创作指南

### 1. 温情治愈类

**核心公式**：
- **主角**：弱势群体（流浪动物、孤独老人、内向小孩）
- **冲突**：与世界的疏离感
- **转折**：微小的善意/陪伴/理解
- **结局**：温暖的连接，不一定是圆满，但要有希望

**情感曲线**：孤独 → 试探 → 信任 → 温暖

**经典套路**：
- 流浪动物被收养（流浪期→幸福期的强烈对比）
- 陌生人的善意（雨天的一把伞、深夜的一碗面）
- 旧物回忆（老照片、旧玩具触发的美好回忆）
- 跨物种友谊（老人与猫、小孩与狗）

**避坑指南**：
- 避免过度煽情，克制比泛滥更有感染力
- 转折要自然，不能为了感动而感动
- 结局可以是 bittersweet（苦乐参半），不一定大团圆

**Prompt 提示**：
- 在画面描述 prompt 中多用"柔和""温暖""自然光""暖色调""金色阳光"等词汇
- 角色表情强调"湿润的眼神""微笑""依偎"等温暖动作

---

### 2. 奇幻冒险类

**核心公式**：
- **主角**：平凡角色获得非凡能力/进入非凡世界
- **冲突**：外部威胁（邪恶势力、自然灾难）或内心恐惧
- **转折**：发现真正的力量来自内心/友谊/勇气
- **结局**：成长与回归，带着改变回到日常生活

**情感曲线**：好奇 → 兴奋 → 恐惧 → 勇气 → 成长

**世界观构建**：
- **魔法规则**：明确且简单的魔法体系（不要复杂）
- **视觉符号**：独特的视觉元素（发光植物、漂浮岛屿、机械生物）
- **文化融合**：可以融合东方/西方神话元素创造新鲜感

**经典套路**：
- 意外获得魔法伙伴（精灵、神兽、机器人）
- 穿越到异世界完成使命
- 小人物拯救大世界
- 寻找失落的神器/记忆

**避坑指南**：
- 世界观不要太复杂，15秒片段讲不清复杂设定
- 魔法要有代价，无敌的主角没有看点
- 奇幻元素要服务于情感，不要为了炫技而奇幻

**Prompt 提示**：
- 在 prompt 中强调"特效粒子""发光""魔法光效""能量流动""透视夸张"
- 场景描述强调"宏大""壮观""神秘"等氛围词

---

### 3. 科幻未来类

**核心公式**：
- **主角**：人机关系中的"人"或觉醒的"机"
- **冲突**：科技与人性的矛盾（孤独、记忆、情感）
- **转折**：发现情感比数据更重要/人性的光辉
- **结局**：开放或反思，留下思考空间

**情感曲线**：疏离 → 好奇 → 冲突 → 理解 → 温情/反思

**科幻元素**：
- **近未来**：AI助手、虚拟现实、基因改造（贴近现实更易共鸣）
- **远未来**：太空殖民、时间旅行、意识上传（需要更多视觉铺垫）
- **赛博朋克**：高科技低生活，霓虹、雨夜、机械义肢

**经典套路**：
- 机器人学习人类情感
- 记忆可以被编辑的世界找回真实
- 末日后的文明重建
- 虚拟与现实的边界模糊

**避坑指南**：
- 科幻设定要服务于情感主题，不要硬科幻
- 避免过度解释技术原理，观众不关心
- 人机关系的核心是"情感连接"而非"技术展示"

**Prompt 提示**：
- 在 prompt 中强调"霓虹""全息投影""金属质感""赛博朋克""冷色调"
- 光影描述强调"霓虹灯反射""雨夜湿润地面""LED光源"

---

### 4. 古风国风类

**核心公式**：
- **主角**：侠客、仙子、书生、妖灵等具有东方美学特征的角色
- **冲突**：情与义的抉择、人与妖的隔阂、宿命与自由的对抗
- **转折**：悟道、放下、牺牲或超越
- **结局**：意境留白，余韵悠长

**情感曲线**：宁静 → 波澜 → 挣扎 → 顿悟 → 超脱

**美学元素**：
- **色彩**：水墨黑白、青花蓝白、朱砂红金、翠竹青
- **意象**：梅兰竹菊、山水云雾、亭台楼阁、古琴长剑
- **意境**：留白、虚实相生、诗情画意

**经典套路**：
- 仙侠情缘（人妖恋、师徒情、前世今生）
- 江湖恩怨（复仇、救赎、快意恩仇）
- 文人雅趣（琴棋书画、诗酒花茶）
- 志怪传奇（聊斋式的鬼怪故事）

**避坑指南**：
- 避免堆砌古风词汇，要有真正的东方美学内核
- 动作设计要有武侠/仙侠的飘逸感，不要现代打斗
- 结尾要留白，给观众想象空间

**Prompt 提示**：
- 在 prompt 中强调"水墨晕染""飘逸""留白""云雾缭绕""古典意境"
- 色彩描述使用"青""墨""朱砂""金""翠"等中国传统色名

---

### 5. 恐怖悬疑类

**核心公式**：
- **主角**：好奇心旺盛或被迫卷入的普通人
- **冲突**：未知的恐惧（鬼怪、阴谋、心理阴影）
- **转折**：真相揭示（往往比鬼怪更可怕的是人心）
- **结局**：开放式或反转，细思极恐

**情感曲线**：平静 → 不安 → 恐惧 → 高潮 → 反转/余悸

**恐怖类型**：
- **心理恐怖**：氛围营造、声音设计、未知恐惧（推荐，成本低效果好）
- **视觉恐怖**：Jump scare、怪物设计（需要高质量美术）
- **悬疑推理**：谜题、线索、反转（需要逻辑严密）

**氛围营造**：
- **光线**：单一光源、大面积阴影、闪烁的灯光
- **声音**：细微的异响、突然的静默、心跳声
- **空间**：封闭空间、迷宫、镜子、窗户

**经典套路**：
- 搬进新家发现秘密（鬼屋、密室、前任租客）
- 循环时间/空间（走不出去的走廊、重复的某一天）
- 镜子中的另一个世界
- 看似正常的人/物逐渐显露异常

**避坑指南**：
- 恐怖要有情感内核（孤独、愧疚、执念），不要纯吓人
- 15秒片段很难营造持续恐怖，建议用悬疑替代
- 反转要合理，不要为了反转而反转

**Prompt 提示**：
- 在 prompt 中强调"阴影""单一光源""暗色调""诡异""不对称"
- 氛围描述强调"寂静""微弱光线""模糊的轮廓""若隐若现"

## 创作流程

### 阶段 1：灵感萃取

从用户输入中提取核心创作元素：

**如果是图片输入**：
- 分析画面中的视觉情绪（光影、色彩、构图传递的氛围）
- 提取可延展的叙事锚点：一个姿态、一个眼神、一个物件、一个环境细节
- 思考"这张图片之前发生了什么？之后会发生什么？"

**如果是文字输入**：
- 识别关键词背后的情感需求（"治愈"=安全感+微小确幸；"悬疑"=信息不对称+紧张感）
- 追问具体性：什么样的治愈？谁的故事？在什么世界？

**如果是视频输入**：
- 分析原片的运动节奏、情绪基调、视觉风格
- 决定是延续、颠覆还是重新诠释

**用户其他输入**：
- 分析用户要求的时间，根据时间规划剧本丰富度

**输出物**：
- 核心情感关键词（1-2个）
- 故事类型判定（温情治愈类 / 奇幻冒险类 / 科幻未来类 / 古风国风类 / 恐怖悬疑类）
- 初步故事梗概（2-3句话）

**验证标准**：
- 用户确认创作方向
- 故事类型已明确

---

### 阶段 2：故事定型

1. 根据阶段 1 确定的故事类型，查阅上方对应的「故事类型创作指南」
2. 按照该类型的核心公式 + 情感曲线 + 经典套路进行创作
3. 同时参考该类型的「避坑指南」避免常见问题

**剧本细化 (Script)**：
- 用户确认梗概后，生成详细构思方案
- **格式要求**：
  - **故事背景**：世界观、时间、地点
  - **角色小传**（使用下方视觉特征清单模板）
  - **分场剧本**：
    - 场次 1：[场景] [时间]
    - 剧情描述...
    - 关键台词...
    - 预计镜头组数量：X 个

**输出物**：
- 完整剧本（含故事背景、角色小传、分场剧本）
- 每场戏标注预计镜头组数量

**验证标准**：
- 剧本逻辑连贯，情感弧线清晰
- 角色视觉特征可执行（有明确的外在锚点）
- 用户确认剧本满意

---

### 阶段 3：分镜拆解

将完整剧本转化为 Storyboard 格式（遵循主流程的 Storyboard 格式规范）。

**操作步骤**：
1. 按场次将剧本拆解为镜头组（每组 ≤ 15 秒）
2. 为每个镜头组编写：画面描述、运镜、音效
3. 标注角色素材引用（@角色）
4. 编制素材清单（角色、道具、音色需求）

**⚠️ 场景描述要求**：
由于不会单独生成场景参考图，每个镜头组的画面描述必须包含**完整的场景环境信息**，作为视频生成 prompt 的直接输入。每个画面描述应涵盖：
- **视觉风格**：如"3D皮克斯动画风格""赛博朋克风格"
- **光线**：光源类型、方向、强度（如"柔和午后阳光从窗帘后洒入"）
- **色彩**：整体色调（如"暖黄色调""灰蓝色冷调"）
- **空间与物件**：具体的环境细节（如"木质地板、米色地毯、窗台绿植"）
- **氛围**：天气、时间、情绪感受（如"阴沉雨天""宁静温暖"）

**输出物**：
- 完整 Storyboard（含全局信息 + 素材清单 + 镜头组列表）

**验证标准**：
- 每个镜头组 ≤ 15 秒
- 镜头组之间逻辑连贯
- 素材清单完整（角色、道具均已列出）
- 一致性标注准确（@角色引用正确）
- 每个镜头组的画面描述包含足够的场景环境信息（光线、色彩、空间、氛围）

## 角色视觉特征清单模板

每个角色必须填写：

| 字段 | 说明 | 示例 |
|------|------|------|
| 姓名/代号 | 角色名称 | 小橘（流浪猫） |
| 外在锚点(2-3个) | 具体视觉特征，将成为一致性关键 | 橘白毛色、左耳缺角、绿色眼睛 |
| 体型比例 | 身体比例描述 | 偏瘦，头身比 1:2.5 |
| 标志性动作 | 用动作展现性格 | 紧张时舔爪子 |
| 核心欲望 | 这个角色想要什么（表面） | 想要一个温暖的窝 |
| 深层需求 | 这个角色真正需要什么（情感内核） | 被接纳和信任 |

## 输出质量标准

一个优秀的创意构思应该让后续环节的生产者（storyboard设计师、美术指导）能够：
1. 清晰理解故事的叙事节奏和情感弧线
2. 提取出需要视觉化的关键元素（角色、场景、道具）
3. 预判技术难点并提前规划解决方案
4. 感受到故事的独特气质和美学方向

## 常见问题处理

**问题：用户想法过于模糊（如"帮我想一个有趣的故事"）**
→ 通过提问缩小范围：目标受众是谁？偏好什么类型？有没有参考作品？
→ 提供 2-3 个不同类型的简短梗概供选择

**问题：用户需求超出时长承载能力（如15秒内讲完复杂故事）**
→ 参考时长-复杂度对照表，建议增加时长或简化情节
→ 给出精简版和完整版两个方案供选择

**问题：用户要求的元素之间存在矛盾（如"欢快的恐怖片"）**
→ 先确认用户真正想要的效果，可能是"黑色幽默"或"反差萌恐怖"
→ 提供该组合的可行方案（如：轻松氛围中突然出现诡异元素）

**问题：剧本中角色过多，难以在短片中展开**
→ 建议合并功能相似的角色
→ 优先保留与核心冲突直接相关的角色

## 示例 Prompt

"基于这张废墟图片，构思一个关于'希望'的短片，时长约 30 秒。我希望是治愈风格。"
