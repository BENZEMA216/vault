# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述

这是一个 Obsidian 知识库，包含 **Creative CoWork** 产品的设计文档和学习笔记。

Creative CoWork 是面向专业创作者（"超创"）的 AI 创作工作台，定位为"Claude Code 的创作版"。

## 目录结构

```
WORK/
├── The Best Idea/     # 核心产品文档
│   ├── 01-04 *.md     # Creative CoWork 设计文档
│   └── resources/     # 相关资源
├── Study/             # 学习笔记和论文
├── User Interview/    # 用户访谈记录
└── DEMO INTRO/        # Demo 相关材料
```

## 核心概念

- **超创**：能用 AIGC 赚钱的专业创作者（目标用户）
- **上下文容器**：聚合项目所有上下文（素材、生成结果、参考、对话历史）
- **AGENT**：自然语言交互，具备 L3-L4 智能（上下文利用 + 多步编排）
- **Studio**：场景化工作空间模板（工具 + 界面 + 配置）
- **Skills**：用户可自由加载的模块化能力
- **GENUI**：基于 skills/上下文动态生成的界面
- **SANDBOX**：代码执行环境，用于编排和执行任务

## 编辑规范

- 所有文档使用中文
- 使用 Obsidian 风格的 Markdown（支持 `[[链接]]` 语法）
- 更新文档时保持现有结构和编号
- 添加新内容时注意与相关章节的交叉引用
- Study 文件夹用于存放论文笔记和学习材料
- **下载的论文 PDF 统一放在 Study/论文 文件夹**
- **User Interview 文件夹中的访谈原文禁止修改**，只能新建文件做总结或分析
