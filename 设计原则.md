# StoryCode Studio - Design Tokens / CSS 样式表

## 概述

本文档定义了 StoryCode Studio 界面设计的样式规范，包括颜色、字体、间距、圆角等设计 Token。

---

## 1. 颜色 (Colors)

### 背景色

| Token | 值 | 用途 |
|-------|-----|------|
| `--bg-primary` | `#0f0f12` | 主背景 |
| `--bg-secondary` | `#16171a` | 侧边栏/面板背景 |
| `--bg-tertiary` | `#0d0d0f` | 标签栏背景 |

### 文字色

| Token | 值 | 用途 |
|-------|-----|------|
| `--text-primary` | `#ffffff` | 主要文字 |
| `--text-secondary` | `#ffffffcc` | 次要文字 (80%) |
| `--text-tertiary` | `#ffffffaa` | 三级文字 (67%) |
| `--text-muted` | `#ffffff66` | 弱化文字 (40%) |
| `--text-disabled` | `#ffffff44` | 禁用文字 (27%) |
| `--text-hint` | `#ffffff33` | 提示文字 (20%) |

### 强调色

| Token | 值 | 用途 |
|-------|-----|------|
| `--accent-primary` | `#22f2ff` | 主强调色 (青色) |
| `--accent-success` | `#4ade80` | 成功/运行中 (绿色) |
| `--accent-info` | `#4fc3f7` | 信息/文件图标 (蓝色) |

### 交互状态

| Token | 值 | 用途 |
|-------|-----|------|
| `--hover-bg` | `#ffffff14` | 悬停背景 (8%) |
| `--active-bg` | `#22f2ff1a` | 激活背景 (10%) |
| `--input-bg` | `#ffffff0a` | 输入框背景 (4%) |
| `--input-border` | `#ffffff14` | 输入框边框 (8%) |

### 边框

| Token | 值 | 用途 |
|-------|-----|------|
| `--border-default` | `#ffffff0d` | 默认边框 (5%) |
| `--border-subtle` | `#ffffff14` | 细微边框 (8%) |

## 6. 组件样式

### Tab 标签页

```
.tab {
  padding: 6px 10px;
  border-radius: 6px 6px 0 0;
  font-size: 11px;
  gap: 6px;
}
.tab--active {
  background: var(--bg-secondary);
  color: var(--text-primary);
}
.tab--inactive {
  background: transparent;
  color: var(--text-tertiary);
}
```

### FileItem 文件项

```
.file-item {
  height: 28px;
  padding: 0 16px;
  gap: 6px;
  font-size: 13px;
}
.file-item--selected {
  background: var(--hover-bg);
}
```

### Input 输入框

```
.input {
  padding: 12px 16px;
  border-radius: 12px;
  background: var(--input-bg);
  border: 1px solid var(--input-border);
  font-size: 13px;
}
```

### Button 按钮

```
.btn {
  padding: 6px 12px;
  border-radius: 6px;
  font-size: 12px;
}
.btn--primary {
  background: var(--active-bg);
  color: var(--accent-primary);
}
.btn--secondary {
  background: var(--hover-bg);
  color: var(--text-secondary);
}
```

### StatusDot 状态指示器

```
.status-dot {
  width: 6px;
  height: 6px;
  border-radius: 50%;
}
.status-dot--active { background: var(--accent-primary); }
.status-dot--running { background: var(--accent-success); }
.status-dot--idle { background: var(--text-disabled); }
```
