## TL;DR（3 个核心判断）

1. **2026 不是融资寒冬，是分化年**：AI infra 在降温，AI 应用层 2025 单年融了 ¥1070 亿、930 家公司、日均 2.6 家拿钱。产品型 AI 创业者反而是红利最大的一侧。

2. **Super-Individual 已经是 2026 的正规叙事**：陈天桥 24 小时给 MiroFish ¥3000 万、Base44 6 个月零融资被 Wix 以 $80M 收购、美国 2025 H1 单人创始公司占比 46.3%。vibe coding 把"1 个人 6 个月做出 PMF"变成常态，第一笔钱的逻辑正在被改写。

3. **对你最现实的融资序列是**：Bootstrapped 3 个月 → 奇绩创坛 → 个人天使（陈天桥 / 王慧文 / 朱啸虎个人）。跳过亲友股权钱，保持 cap table 干净；用 Wiki MCP Server 作为"可演示资产"而不是空想 thesis。

### 🟠 TIER 4: 亲友轮（只走借款，避免股权）

**两种合法路径**:

| 形式 | 特点 | 风险 |
|---|---|---|
| **借款（推荐）** | 固定本金 + 利息 4-8%，不参与经营 | 低，但必须签《借款合同》明确条款 |
| **股权投资** | 参与盈亏，签《投资协议》| 高，cap table 变乱 |

**三条致命红线**:
- ❌ **不要平分股权**（50/50 或 33/33/33 是最不稳定结构，最高法多次判例）
- ❌ **不要拿超过 ~30 个亲友的钱**（可能触发"非法集资"刑事红线）
- ❌ **不要没签书面协议**（法院按合伙处理，亏了要赔）

**建议**: 只走"借款"路径，不让亲友持股，保持 cap table 干净进入奇绩/VC。

## 四、MiroFish 为什么能 24 小时融到？（解构 anomaly）

**4 个独立低概率事件的乘积**:

1. **冷启动 = 全球可见的客观信号**：GitHub Trending #1，几天 20k★。投资人无法 fake 的"市场已投票"信号，省去尽调全流程
2. **作品 + 创始人故事高度耦合**：20 岁、大四、10 天 vibe coding、做红楼梦后 40 回——一个 narrative 三个梗
3. **陈天桥这类 super-LP 绕开 IC 流程**：他不需要合伙人会议，要的就是"first call rights on solo viral founders"
4. **方向命中个人 thesis**：multi-agent + 群体智能 + 脑科学外溢，正好是天桥脑科学研究院主线
5. **"deep incubation"而非纯股权交易**：估值条款模糊，速度优先

**可复制性低，但启示意义高**：vibe coding 时代的核心货币 = **"可被截屏的客观市场信号 + 一句话讲完的 founder story"**。Demo / ARR 截图 / GitHub 星标 / Twitter 病毒都是同一种东西。

## 五、典型融资时间线

### 标准 VC 流程（3-5 个月）
```
决定融资 → 准备 deck/demo    : 1-2 周
Outreach → 首次见面          : 2-4 周
多轮会谈 → 第一份 TS         : 3-6 周
TS → SPA → 工商变更          : 4-8 周
工商变更 → 资金到账          : 1-4 周
总计                         : 3-5 个月
```

### 个人天使 / 超角色流程（24h - 2 周）
```
陈天桥模式：看 GitHub 数据 + demo video → 24h 打款
王慧文模式：个人 LP 出资 → 1-2 周
```

### 奇绩创坛流程（固定 batch）
```
申请开放          : 常年
截止             : 12 月底（延期可）
初筛             : 12 月-2 月
面试             : 3-6 月
创业营           : 3-6 月（或 9-12 月）
Demo Day         : 6 月 / 12 月
Demo Day 后 3 天 : 40%+ 团队拿到下一轮 TS
```

## 九、关于香港赴港的融资影响

### 正面

- **HKSTP Incubation Programme**: 最高 HK$1.29M（约 US$165K）**非股权型**
- **HKSTP × AWS AI Co-incubation**（2025 新启）: AI-first, Seed-A 阶段
- **Cyberport**: HK$500K 非股权，偏 Fintech
- **HK AI Innovation Fund**: HK$10B 政府基金
- **落户补贴** + 税率优势（利得税 16.5% 两级制）

### 负面

- 奇绩入营**必须能去北京 3 个月**（香港只是宣传/面试节点）
- 国内主流 VC 对港注册公司尽调更麻烦
- 人脉/曝光远不如北京/深圳

### 权衡

- **创业早期**: 北京/深圳 > 香港（奇绩 + 真格 + 陈天桥都在 mainland）
- **规模化后**: 香港 > 大陆（税率 + 国际市场 + 资金出入）
- **折中方案**: 北京注册 + 香港研发中心 + 未来移架构
