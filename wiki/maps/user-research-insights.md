## 共性痛点（4/4 团队一致提及）

### 1. 角色一致性

所有团队的**第一痛点**。同一角色在不同镜头中外观不一致：衣服穿错、面部变形、多角色混淆。

- LSW：大量垫图 + 固定 prompt 模板 + 多次生成筛选 + 后期修图
- 三界动画：多角色场景用 Banana 模型处理，3人以上场景尤其困难
- 艺柏AILab：真人剧需换脸工具（liblib/RunningHub），说话时脸会变
- 福州畅读：底模太少导致所有帅哥长一样，多人一致性差

> "短剧最难做的是初恋的情——扇巴掌、哭、打斗、人物不变形，你这环节生成不出来就得重新设计。" —— 福州畅读

### 2. 上下文与资产分散

素材散落在多个平台和工具之间，缺乏统一的项目级管理。

- LSW：素材散落多处，项目文件组织混乱，版本管理困难
- 三界动画：审核在微信群、资产在 NAS、制作在不同平台
- 艺柏AILab：飞书文档 + 分秒帧 + 多个生成工具，小团队互相补位
- 福州畅读：自研管理系统（角色库、分镜管理）来解决此问题

> "给我一个地方，把素材、参考、生成结果都放在一起，能看到整个项目的全貌。" —— LSW

### 3. 抽卡效率低

大量时间花在"筛选"而非"生成"，生成 → 筛选 → 再生成的循环消耗巨大。

- 三界动画：单镜头抽 30-100 次常见
- 艺柏AILab：5-10 次不行就改分镜（不死磕策略）
- 福州畅读：精品剧自动化程度 < 30%

### 4. 工具碎片化

没有一站式解决方案，必须多工具组合。

- 生图：即梦 + Midjourney + Nano + Stable Diffusion
- 生视频：可灵 + 智影 + 海螺 + Sora2 + Runway
- 剪辑：剪映 + PR + NUKE
- 配音：豆包 TTS + 真人 + 嘎嘎

> "没有一个工具能一站式解决，必须多工具组合。" —— 福州畅读

## 差异化洞察

### 一人全流程 vs 分工制

三界动画的关键转型：从分段制作（分镜、生图、剪辑各有专人）到一人从头做到尾。**分工反而增加沟通成本**，出错率更低，反馈时间更少。这验证了 Creative CoWork "一个人 + Agent" 的产品模式。

### 剪辑前置思维

艺柏AILab 的核心理念：所有设计都基于 AI 能力边界，提前考虑实现方式。分镜时就决定用什么生成方式。音乐也要前置定——有了音乐，画面构图灵感更清晰。

### 出海放大效应

福州畅读的战略："田忌赛马"——用上等马打中等马。海外赚美金，同样 5% ROI 放大 7 倍。单部剧跑出约 300 万美刀。转绘模式降低门槛：已有分镜和配音，只需做画面转换。

### 质量分级生产

三界动画的三级体系：番剧（最高质量，单镜头审核）→ 标准漫剧（中等）→ 推文/解说（跑量）。质量递减规律：前 5 集质量最高，后期越来越松。

## 超创用户画像

综合 4 个团队访谈 + 竞品分析，超创（[[concepts/super-creators]]）的画像：

| 维度 | 特征 |
|------|------|
| **身份** | 能用 AIGC 赚钱的专业创作者，不是业余爱好者 |
| **工作模式** | 一人全链路或小团队（3-5人），导演+Agent 模式 |
| **核心诉求** | 效率 > 质量上限，时间成本 > 工具成本 |

| **决策方式** | 分步确认 > 一键全自动，关键节点需要人工介入 |
| **痛点优先级** | 角色一致性 > 上下文管理 > 抽卡效率 > 工具碎片化 |
| **收入来源** | 品牌广告、平台分成、出海变现、成制服务 |

## 产品机会

1. **统一上下文容器**：解决资产分散问题，项目级角色/场景/素材知识库
2. **智能筛选 Agent**：不是一键全自动，而是分步骤批量生成 + AI 辅助筛选
3. **创意循环编排**：发散（多种可能性）→ 收敛（人工确认）→ 执行（批量生成）
4. **全流程工具集成**：通过 Skills 系统统一接入多个生成工具，减少切换成本
5. **出海基础设施**：多语种配音、文化适配、海外平台分发
