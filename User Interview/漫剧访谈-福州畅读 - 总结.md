## 一、公司概况

### 基本信息
- **总部**：福州
- **核心业务**：短剧出海，真人短剧 + AI 漫剧
- **核心资产**：自有短剧 IP 版权、海外分发平台
- **战略定位**："田忌赛马"——用上等马打中等马

### 业务模式
- **主线**：将自有真人短剧转绘成日韩风格动画出海
- **副线**：小说解说漫（待主线稳固后再做）
- **不做**：沙雕漫、PPT 漫（毛利低、没长期价值）

### 为什么选择出海
- 海外赚美金，同样 5% ROI 放大 7 倍
- 全球 66 亿人（除中国 14 亿），英语覆盖 10 亿+
- 目标市场：英五国、日韩、东南亚

### 代表成绩
- 阿里巴巴海外平台漫剧排名 Top 1
- 单部剧跑出约 300 万美刀

## 二、团队组织

### 核心分工

| 角色 | 人数/组 | 职责 |
|------|---------|------|
| 分镜导演 | 1 | 剧本拆解、人物定妆、场景设计、统筹 |
| 抽卡师/分镜设计师 | ≤10 | 批量生图、调整提示词 |
| 剪辑 | 中台共用 | 粗剪、精剪、物料组装 |
| 配音 | 中台共用 | AI 配音 / 真人配音 |

### 协作方式
- **坐在一起，面对面沟通**，不发消息
- 每个导演有自己的情感风格，不能换人
- 自研 AI 制作管理系统（角色库、分镜管理、批量生成）

> "10 个抽卡师，10 种导演风格，上家接上家你都衔接不起来。"

## 三、产品类型与成本

### 成本分级

| 品质等级 | 单分钟成本 | 特点 |
|----------|-----------|------|
| 低端漫剧 | 300-400 元 | 不做衔接/转场 |
| 精品漫剧 | 1000-1500 元 | 做转场、特写、上下镜衔接 |
| 仿真人剧 | 2000+ 元 | 手搓为主，AI 生成不出来 |

### 整体成本结构
- **一部剧时长**：60-100 分钟
- **制作成本**：7-10 万元（含人工）
- **制作周期**：2 周 ~ 1 个月
- **成本大头**：人工（Token 不贵）

## 四、制作流程

### 完整流程
```
真人剧剧本 → 剧本拆解 → 人物定妆 → 场景设计 → 分镜设计 → 生图抽卡 → 生视频 → 对口型 → 后期剪辑 → 配音 → 交付
```

### 各环节详情

#### 1. 剧本拆解
- **来源**：自有真人剧剧本（已有分镜），或新剧本
- **产出**：分镜脚本（景别、分镜号、画面描述）
- **AI 能做**：辅助拆大纲
- **人工必须**：核心剧情细化、卡点设计、出量镜头识别
- **耗时**：约 1 周（含定妆）

#### 2. 人物定妆
- **产出**：三视图（正/侧/背）+ 多套服装版本
- **工具**：即梦（漫画）、可灵/海螺（真人）
- **痛点**：人物比例失调、衣服反穿、底模太少
- **审核**：多组导演共同审核

#### 3. 场景设计
- **产出**：主要场景参考图（多角度）
- **工具**：即梦
- **痛点**：场景同质化（宴会=吊顶+圆桌）、视角单一

#### 4. 分镜设计
- **产出**：关键帧提示词 + 视频提示词
- **分工**：导演定大框架，抽卡师做细化
- **特点**：抽卡师有一定创作空间

#### 5. 生图抽卡
- **方式**：批量生成 → 人工筛选
- **策略**：5-10 次抽不中 → 换模型 → 改分镜
- **自动化程度**：精品剧 < 30%

#### 6. 生视频
- **方式**：单图生成 / 首尾帧
- **首尾帧使用**：特别重要的镜头（成本翻倍）
- **痛点**：运镜不可控、动作不精准、多人交互难

#### 7. 对口型
- **工具**：拍拍我等第三方
- **标准**：只求音画同步，不求嘴型精确

#### 8. 后期剪辑
- **工具**：剪映团队版
- **内容**：物料梳理、上下镜衔接、口型裁切

#### 9. 配音
- **高质量**：真人配音（海外需 2 周+）
- **低质量**：自研 AI 配音系统（支持多语种情绪）

## 五、使用的工具

### 核心工具矩阵

| 环节 | 工具 | 选择原因 |
|------|------|----------|
| 漫画人物 | 即梦 | 风格好，但有过拟合问题 |
| 真人皮肤 | 可灵/海螺 | 质感好，不磨皮 |
| 生视频 | 即梦/可灵/海螺 | 各有优劣 |
| 换脸 | liblib、RunningHub | 局部修复 |
| 对口型 | 拍拍我 | 音画同步 |
| 剪辑 | 剪映团队版 | 协作方便 |
| 配音 | 自研 AI / 真人 | 海外本地化 |
| 管理 | 自研系统 | 角色库、分镜管理 |

### 多工具组合问题
> "没有一个工具能一站式解决，必须多工具组合。"

## 六、核心痛点

### 1. 人物相关
| 痛点 | 表现 |
|------|------|
| 底模太少 | 所有帅哥/老人长一样 |
| 比例失调 | 头大身小、身材固定 |
| 多人一致性差 | 衣服乱穿、特征混淆 |
| 微表情难控 | 性格表达靠服化道 |

### 2. 场景相关
| 痛点 | 表现 |
|------|------|
| 场景同质化 | 宴会=吊顶+圆桌+香槟 |
| 视角单一 | 走廊只有左/右两个角度 |
| 人物像贴图 | 无空气感、不融入场景 |

### 3. 技术相关
| 痛点 | 表现 |
|------|------|
| 光影逻辑错 | 人自带光效、模型改光 |
| 透视/景深问题 | 焦点对错、比例失真 |
| 运镜不可控 | 左右概念理解差 |
| 动作不精准 | 手放桌上变抬起放下 |

### 4. 出海相关
| 痛点 | 表现 |
|------|------|
| 审核尺度 | 国内严格，保安服装都 pass |
| 多环节审核不一致 | 生图开了，视频卡了 |
| 配音周期长 | 海外真人配音需 2 周+ |

## 七、质量评判标准

### 底层要求（必须达标）
1. 人物一致性
2. 空间一致性（透视正确、景深合理）
3. 美术风格一致性
4. 光影逻辑正确

### 进阶要求
1. 剧情连贯
2. 镜头衔接自然
3. 人物有情感表达

### 漫剧 vs 真人剧区别
> "真人剧多一个'看起来像假的'问题，其他都一样。"
> "暗色调更容易过关，亮色调会暴露硅胶感。"

## 八、对工具的期望

### 最想要的功能
1. **人物底模更丰富**：不要所有帅哥都长一样
2. **场景多样性**：不要所有宴会都有吊顶
3. **多人场景一致性**：衣服不要乱穿
4. **人物融入场景**：不要像贴图
5. **出海尺度放开**：针对海外业务开白名单

### 理想状态
> "我只填一句话内容，帮我生成 5 个视频，我挑一个就行。不想不停改提示词。"

## 九、与 Creative CoWork 的关联

### 痛点匹配

| 访谈痛点 | Creative CoWork 方案 |
|----------|---------------------|
| 资产分散在多平台 | 上下文容器统一管理 |
| 人物/场景同质化 | 项目级资产库 |
| 抽卡效率低 | AGENT 智能筛选 |
| 多人协作靠面对面 | 内置协作流程 |
| 出海配音难 | Skills 集成多语种 TTS |

### 关键洞察

1. **出海是高利润赛道**
   - 同样工作量，收益放大 7 倍
   - 验证了"全球化内容创作"的价值

2. **转绘模式降低门槛**
   - 已有分镜（真人剧导演做过）
   - 已有配音（真人剧配过）
   - 只需做画面转换

3. **导演权威不可替代**
   - 分镜设计、卡点选择需要人类审美
   - 抽卡师有执行空间，但大方向靠导演

4. **工具碎片化是普遍问题**
   - 多工具组合 = 多次切换 = 效率损失
   - 自研系统是大公司的选择

## 十、值得注意的细节

1. **坐一起比发消息效率高**：短视频行业是项目制，标准流程不存在
2. **前 10 集决定生死**：平台算法驱动，开头必须精细
3. **配音先于画面**：配音时长决定画面时长
4. **暗色调更安全**：噪点遮瑕疵，亮色暴露硅胶感
5. **出海审核多环节不一致**：每个环节都要单独开白名单
