## 四、制作流程

### 完整流程
```
剧本 → 人工拆分镜 → 定人设 → 定美术风格 → 定场景 → 生图 → 静帧预演 → 甲方确认 → 生视频 → 剪辑调色 → 交付
```

### 核心理念
> **"剪辑前置"**：所有设计都要基于 AI 能力边界，提前考虑实现方式

### 各环节详情

#### 1. 拆分镜（人工核心）
- **工具**：人脑 + 电影参考网站
- **产出**：画面内容、景别机位、运镜描述
- **为什么不用 AI**：AI 不懂视听语言、不懂高潮铺垫
- **耗时**：一部剧约 1 天

> "AI 拆的剧本效果不好，它不懂什么是高潮，不懂怎么铺垫。"

#### 2. 定人设
- **流程**：与甲方沟通 → 生成形象 → 确认
- **产出**：角色定妆照
- **耗时**：一下午

#### 3. 定美术风格
- **来源**：Midjourney、电影截图网站
- **方式**：测试垫图效果
- **耗时**：一下午

#### 4. 定场景
- **工具**：Midjourney、即梦
- **用途**：给 AI 垫图保持一致性

#### 5. 生图
- **主力工具**：Midjourney（场景/风格）、即梦（人物）、Nano（三视图/改图）
- **痛点**：即梦过拟合（多次编辑后图片"浮粉"）
- **抽卡策略**：5-10 次不行就改分镜

#### 6. 静帧预演
- **工具**：剪映/PR
- **作用**：给甲方确认，减少视频返工

#### 7. 生视频
- **主力工具**：可灵、海螺、VO3、Sora2（最便宜）
- **痛点**：即梦/海螺会改光

#### 8. 剪辑调色
- **调色次数**：3 次（生图时、预演时、成片时）
- **原因**：不同模型颜色/光线不一致

## 五、使用的工具

### 核心工具矩阵

| 环节 | 工具 | 选择原因 |
|------|------|----------|
| 人设生成 | 即梦 | 亚洲人/中国人效果好 |
| 场景/风格 | Midjourney | 质感强 |
| 三视图/改图 | Nano | 质感好、稳定 |
| 生视频 | 可灵/海螺/VO3 | 不改光 |
| 生视频（低成本） | Sora2 | 最便宜（0.1-0.2元/次） |
| 换脸 | liblib、RunningHub | 局部修复 |
| 配音 | 嘎嘎、真人（60元/分钟） | 情绪更好 |
| 音乐 | Sono | AI 作曲 |
| 项目管理 | 文创AI、飞书 | 批量生成、协作 |
| 视频审阅 | 分秒帧 | 甲方沟通 |

### 工具评价
- **即梦**：人物好，但过拟合问题严重
- **Nano**：质感好，三视图稳定
- **可灵/VO3**：不改光
- **即梦/海螺**：会改光，调色麻烦
- **Sora2**：便宜，漫剧首选

## 六、核心痛点

### 1. 生图效率
| 痛点 | 表现 |
|------|------|
| 大场景不好出 | 训练集缺乏 |
| 即梦过拟合 | 多次编辑后"浮粉" |
| 人物方向不可控 | 左右概念理解差 |

### 2. 模型差异
| 痛点 | 表现 |
|------|------|
| 颜色/光线不一致 | 需要 3 次调色 |
| 改光问题 | 即梦/海螺会改光 |

### 3. 人物一致性
| 痛点 | 表现 |
|------|------|
| 真人剧难保持 | 需要换脸工具 |
| 说话时脸变 | 演技好但脸变了 |

### 4. 甲方沟通
| 痛点 | 表现 |
|------|------|
| 效果达不到预期 | 反复抽卡/改分镜 |
| 产品要求精确 | AI 细节不可控 |

## 七、质量评判标准

### 核心认知
> **"故事 50 分 + 视听 10 分 = 合格；故事 50 分 + 视听 20 分 = 优秀"**

### 漫剧标准
1. **爽**：从第一秒开始有信息传递
2. **人物一致**：没有明显穿帮
3. **音画同步**：声音符合画面
4. **叙事连贯**：头 30 集尤其重要

### 真人剧标准
1. 以上全部 +
2. **真实感**：不能有硅胶感
3. **演技**：表情到位

## 九、与 Creative CoWork 的关联

### 痛点匹配

| 访谈痛点 | Creative CoWork 方案 |
|----------|---------------------|
| 工具分散 | Skills 统一接入 |
| 调色繁琐 | 自动化后处理 |
| 甲方沟通成本高 | 内置审阅流程 |
| 分镜依赖人工 | AGENT 辅助（但不能替代） |

### 关键洞察

1. **剪辑前置是共识**
   - 所有设计都要考虑 AI 能力边界
   - 分镜时就决定用什么生成方式

2. **音乐要前置定**
   - 有了音乐，画面构图灵感更清晰
   - 音乐节奏决定分镜节奏

3. **分镜不能全靠 AI**
   - AI 不懂高潮铺垫
   - AI 不懂视听语言
   - 人类导演经验不可替代

4. **漫剧必须走量**
   - 100 分钟/月是门槛
   - 补贴/合作机会需要产能证明

5. **小团队的生存之道**
   - 分工不明确，互相补位
   - 成制为主，降低风险
   - 等时机成熟再做自有 IP

## 十、值得注意的细节

1. **真人配音 60 元/分钟**：比想象中便宜
2. **抽卡 5-10 次不行就改分镜**：不死磕
3. **静帧预演是必须的**：减少视频返工
4. **3 次调色**：生图、预演、成片各一次
5. **北京人力贵**：产教融合是出路，但培训周期 4 个月
